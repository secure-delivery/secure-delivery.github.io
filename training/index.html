Course Introductions

The Secure Delivery Academy has courses covering every aspect of software product security across all your delivery roles and we can create custom self-paced content tailored for your organisation.

Introduction to Software Product Security

Focused on the fundamentals of securing modern digital product delivery, our foundational training covers a broader topic than what's usually termed application security.

This course is built on industry standards and the product of our work within OWASP to define a security curriculum globally for industry and academia.

Roles: All delivery roles

Self-paced Online

Agile Threat Modelling

It's an essential capability for a delivery team to be able to think about the ways their software product could be compromised and capture actionable work that addresses these threats.

Nowadays, this needs to be done in an agile, evolutionary way. This course guides a team through an Agile Threat Modelling session and introduces the Elevation of Privilege and OWASP Cornucopia card games.

Roles: All delivery roles

Interactive Workshop

Secure Coding

73% of all software security vulnerabilities are caused by implementation defects – bugs in code. Trying to catch them once they've been created with security testing tools leads to an endless backlog of fixes that never get done.

Get ahead of the problem with this hands-on, secure coding course that prevents the OWASP Top 10 most common vulnerabilities found in software across all major languages and frameworks.

Roles: Software Developers

Interactive Workshop

OWASP Top 10 for Decision Makers

The OWASP Top 10 is a regularly updated list of the highest risk software product security vulnerabilities based on industry data. Everyone involved in delivery benefits from understanding these risks.

With clear, non-technical explanations of the security vulnerability types and accompanying real-world case studies of serious data breaches caused by them, this course is ideal for product roles at all levels.

Roles: Product decision makers

Self-paced Online

Implementing a Software Security Programme

A high-performing software delivery organisation needs a high-performing approach to security. Our product security framework is based on capabilities derived from the regulatory frameworks and industry standards that apply to your business.

This course is based on our real-world implementation of security programmes that work locally and globally and is ideal for anyone with accountability for overall product delivery.

Roles: Delivery organisation leaders

Self-paced Online

Quality Control for Digital Product Delivery

In effective quality assurance, quality control tooling provides observability into built products and feedback for process improvement. Security test tooling is a crowded marketplace, however, with a steep price tag for mistaken purchases.

Get the understanding you need, together with benchmarking data, across the entire software security tooling landscape with this vendor-agnostic and objective course.

Roles: Delivery organisation leaders
